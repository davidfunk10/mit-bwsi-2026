# Motherboard

The motherboard is the main circuit board of a computer.  
It connects all hardware components together.

---

## Main Components

### CPU Socket
Holds the Central Processing Unit.

### RAM Slots
Where system memory modules are installed.

### Chipset
Controls communication between the CPU, memory, and peripherals.

### PCIe Slots
Used for expansion cards such as:
- graphics cards
- network cards
- sound cards

### Storage Connectors
Allow storage devices to connect.

Examples:
- SATA ports
- M.2 slots

### Power Connectors
Provide power to the motherboard and CPU.

---

## Functions of the Motherboard

- Connects all computer components
- Allows communication between hardware
- Distributes power
- Hosts firmware such as BIOS or UEFI
