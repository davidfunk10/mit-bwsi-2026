# Basic Linux Commands

Linux uses a command-line interface to interact with the system.

---

## Navigation

pwd  
Print the current directory.

cd  
Change directories.

Example:
cd Documents

ls  
List files in the current directory.

---

## File Management

touch filename  
Creates an empty file.

mkdir foldername  
Creates a directory.

rm filename  
Deletes a file.

cp source destination  
Copies files.

mv oldname newname  
Moves or renames files.

---

## Viewing Files

cat file.txt  
Displays file contents.

less file.txt  
Displays file contents one screen at a time.

---

## Help Commands

man command  
Shows the manual page for a command.

Example:
man ls
