# Types of Storage

Storage devices keep data **even when power is turned off**. This is called **non-volatile storage**.

---

## Hard Disk Drive (HDD)

A hard disk drive stores data using **spinning magnetic disks**.

### Characteristics
- large storage capacity
- relatively inexpensive
- slower than solid state drives
- mechanical parts can wear out over time

### Common Uses
- large file storage
- backups
- older computers

---

## Solid State Drive (SSD)

A solid state drive stores data using **flash memory** instead of spinning disks.

### Characteristics
- much faster than HDDs
- no moving parts
- more durable
- lower latency
- uses less power

### Important Note
SSDs can connect using different interfaces, such as:
- **SATA**
- **PCIe/NVMe**

---

## NVMe SSD

An **NVMe SSD** is a type of solid state drive that uses the **NVMe protocol over PCIe**.

### Characteristics
- extremely fast data transfer speeds
- lower latency than SATA SSDs
- commonly used in modern computers
- often installed in **M.2 slots** on the motherboard

### Important Distinction
- **PCIe** = hardware connection
- **NVMe** = communication protocol

---

## USB Flash Drive

A USB flash drive is a portable storage device that uses **flash memory**.

### Characteristics
- small and portable
- connects through USB ports
- commonly used for transferring files

### Common Uses
- file transfer
- backups
- portable storage

---

## Optical Storage

Optical storage uses **laser technology** to read and write data on discs.

### Examples
- CD (Compact Disc)
- DVD (Digital Versatile Disc)
- Blu-ray Disc

### Characteristics
- removable media
- slower than modern storage devices
- mostly used for media or archival storage
