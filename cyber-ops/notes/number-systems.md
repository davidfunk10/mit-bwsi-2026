# Number Systems

Computers store and process information using different number systems. The most common systems are Binary, Decimal, Hexadecimal, and Octal.

---

## Binary (Base 2)

Binary uses only two digits:

0 and 1

Each digit is called a **bit**.

Example:
1011₂

Calculation:
1×2³ + 0×2² + 1×2¹ + 1×2⁰  
= 8 + 0 + 2 + 1  
= 11 (decimal)

Binary is used internally by computers because electronic circuits have two states:
- ON
- OFF

---

## Decimal (Base 10)

The number system humans normally use.

Digits:
0–9

Example:
347₁₀

Calculation:
3×10² + 4×10¹ + 7×10⁰

---

## Hexadecimal (Base 16)

Uses 16 symbols:

0–9 and A–F

Values:
A = 10  
B = 11  
C = 12  
D = 13  
E = 14  
F = 15

Example:
2F₁₆

Calculation:
2×16¹ + 15×16⁰  
= 32 + 15  
= 47 (decimal)

Hexadecimal is commonly used in:
- memory addresses
- machine code
- color values

---

## Octal (Base 8)

Uses digits:

0–7

Example:
17₈

Calculation:
1×8¹ + 7×8⁰  
= 8 + 7  
= 15 (decimal)

Octal was historically used in early computing systems.

---

## Why Multiple Number Systems Exist

Binary is easiest for computers to process.

Hexadecimal and Octal are used because they provide a **shorter, more readable representation of binary data**.

Example:

Binary:
11111111

Hex:
FF
