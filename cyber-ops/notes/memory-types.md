# Types of Memory

Computer memory temporarily stores data that the CPU is currently using.

---

## RAM (Random Access Memory)

Temporary working memory.

Characteristics:
- volatile (data lost when power is off)
- very fast access
- used for running programs

---

## ROM (Read Only Memory)

Stores permanent instructions.

Characteristics:
- non-volatile
- used for firmware

---

## Cache Memory

Small amount of very fast memory located near the CPU.

Levels:
- L1 cache
- L2 cache
- L3 cache

Used to store frequently accessed data.

---

## Virtual Memory

A portion of disk storage used as additional RAM when system memory is full.
